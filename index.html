1. Introducción a Python

¿Qué es Python?

Python es un lenguaje de programación de alto nivel, interpretado y de propósito general. Se usa en desarrollo web, ciencia de datos, automatización de tareas, inteligencia artificial, y más.

Ventajas de Python

  • Sintaxis sencilla y fácil de leer.
  • Gran comunidad y muchas bibliotecas disponibles.
  • Portabilidad: funciona en Windows, macOS, y Linux.

Objetivos del Curso

  • Aprender los conceptos básicos de Python.
  • Escribir y ejecutar scripts simples.
  • Entender la lógica de programación básica.

2. Instalación de Python

Descargar Python

Visita python.org y descarga la última versión estable para tu sistema operativo.

Instalación en Windows

  1. Ejecuta el instalador.
  2. Asegúrate de seleccionar "Add Python to PATH".
  3. Sigue las instrucciones del instalador.

Instalación en macOS

Abre la terminal y usa Homebrew: brew install python

Verificar la Instalación

Abre la terminal (cmd en Windows) y escribe python --version o python3 --version. Deberías ver la versión de Python instalada.

3. Primeros Pasos con Python

Tu Primer Programa en Python

Abre un editor de texto (Notepad, VS Code, PyCharm) y escribe el siguiente código:

print("¡Hola, Mundo!")

Guarda el archivo como hola_mundo.py.

Ejecutar el Programa

Abre la terminal, navega al directorio donde guardaste el archivo y escribe python hola_mundo.py o python3 hola_mundo.py. Deberías ver ¡Hola, Mundo! en la terminal.

print("Bienvenidos al curso de Python")\nprint("¡Vamos a aprender juntos!")

4. Variables y Tipos de Datos

¿Qué es una Variable?

Una variable es un espacio en memoria donde se almacena un valor. Ejemplo:

x = 5

Tipos de Datos Básicos

  • Enteros (int): x = 5
  • Flotantes (float): y = 3.14
  • Cadenas de texto (str): nombre = "Juan"
  • Booleanos (bool): es_mayor = True

Imprimir Variables

nombre = "Juan"\nedad = 25\nprint("Nombre:", nombre)\nprint("Edad:", edad)
pi = 3.14159\nes_estudiante = False\nprint("Valor de pi:", pi)\nprint("Es estudiante:", es_estudiante)

5. Operadores Básicos

Operadores Aritméticos

  • Suma: +
  • Resta: -
  • Multiplicación: *
  • División: /
a = 10\nb = 5\nprint(a + b)  # Resultado: 15\nprint(a - b)  # Resultado: 5\nprint(a * b)  # Resultado: 50\nprint(a / b)  # Resultado: 2.0

Operadores de Comparación

  • Igual a: ==
  • Diferente de: !=
  • Mayor que: >
  • Menor que: <
x = 10\ny = 5\nprint(x == y)  # Resultado: False\nprint(x != y)  # Resultado: True\nprint(x > y)   # Resultado: True\nprint(x < y)   # Resultado: False
z = 15\nprint(x <= z) 
                Resultado: True\nprint(y >= z)  # Resultado: False

6. Estructuras de Control

Condicionales (if, elif, else)

Permiten tomar decisiones en el código.

edad = 20\nif edad >= 18:\n    print("Eres mayor de edad")\nelse:\n    print("Eres menor de edad")

Ejemplo con elif:

nota = 85\nif nota >= 90:\n    print("Excelente")\nelif nota >= 70:\n    print("Bueno")\nelse:\n    print("Necesita mejorar")

Bucles (for, while)

Bucle for:

for i in range(5):\n    print(i)

Bucle while:

contador = 0\nwhile contador < 5:\n    print(contador)\n    contador += 1
for fruta in ["manzana", "banana", "cereza"]:\n    print(fruta)

7. Funciones

¿Qué es una Función?

Un bloque de código que realiza una tarea específica. Se puede reutilizar en diferentes partes del programa.

Definir una Función

def saludar(nombre):\n    print("Hola,", nombre)

Llamar a una Función

saludar("Juan")

Parámetros y Argumentos

Parámetros: Variables que se definen en la función.

Argumentos: Valores que se pasan a la función.

def sumar(a, b):\n    return a + b\n\nresultado = sumar(3, 5)\nprint("Resultado:", resultado)
def multiplicar(x, y):\n    return x * y\n\nprint("Multiplicación:", multiplicar(4, 5))

8. Listas y Diccionarios

Listas

Una colección ordenada de elementos.

frutas = ["manzana", "banana", "cereza"]\nprint(frutas[0])  # Resultado: manzana

Agregar Elementos a una Lista

frutas.append("naranja")\nprint(frutas)  # Resultado: ["manzana", "banana", "cereza", "naranja"]

Eliminar Elementos de una Lista

frutas.remove("banana")\nprint(frutas)  # Resultado: ["manzana", "cereza", "naranja"]

Diccionarios

Colección de pares clave-valor.

estudiante = {\n    "nombre": "Ana",\n    "edad": 22,\n    "curso": "Matemáticas"\n}\nprint(estudiante["nombre"])  # Resultado: Ana

Agregar un Nuevo Par Clave-Valor

estudiante["universidad"] = "U de Chile"\nprint(estudiante)

Eliminar un Par Clave-Valor

del estudiante["curso"]\nprint(estudiante)
colores = ["rojo", "verde", "azul"]\ncolores[1] = "amarillo"\nprint(colores)  # Resultado: ["rojo", "amarillo", "azul"]

9. Manejo de Archivos

Abrir y Leer un Archivo

archivo = open("datos.txt", "r")\ncontenido = archivo.read()\nprint(contenido)\narchivo.close()

Leer Línea por Línea

archivo = open("datos.txt", "r")\nfor linea in archivo:\n    print(linea.strip())\narchivo.close()

Escribir en un Archivo

archivo = open("datos.txt", "w")\narchivo.write("Hola, archivo!")\narchivo.close()

Agregar Contenido a un Archivo Existente

archivo = open("datos.txt", "a")\narchivo.write("\nNueva línea de texto.")\narchivo.close()
with open("datos.txt", "r") as archivo:\n    for linea in archivo:\n        print(linea.strip())
